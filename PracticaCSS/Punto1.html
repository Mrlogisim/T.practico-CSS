¿Que es CSS?

Css es un lenguaje de hojas de estilos creado que permite modificar el aspecto o presentación de las paginas hechas con html y Xhtml.

Historia

CSS, que significa "Cascading Style Sheets" (Hojas de Estilo en Cascada), fue creado por el World Wide Web Consortium (W3C) como un estándar para definir la presentación y el diseño de documentos HTML. La primera especificación de CSS, conocida como CSS nivel 1 (CSS1), fue publicada por el W3C en diciembre de 1996.

Desde entonces, CSS ha experimentado varias revisiones y actualizaciones. Estas revisiones han introducido nuevas características, mejoras en la sintaxis y correcciones de errores. Las versiones más notables incluyen:

  • CSS nivel 2 (CSS2): Publicado en mayo de 1998.
  • CSS nivel 2.1 (CSS2.1): Publicado como una revisión de CSS2 en junio de 2011.
  • CSS nivel 3 (CSS3): Introducido como una serie de módulos independientes que se han ido publicando a lo largo del tiempo, a partir de 1999 y hasta la actualidad. CSS3 ha introducido muchas nuevas características y capacidades, como sombras, gradientes, transformaciones, animaciones, y mucho más.
